# Co-DesigndeAplicativos-G6
Projeto de Co-Design de Aplicativos - Insper | 2° Semestre - Grupo 6

No trabalho aqui desenvolvido, foi criado um aplicativo de marketplace para a compra e venda de produtos agrícolas, com o objetivo de ajudar pequenos e médios produtores a venderem seus bens de forma mais fácil, tranquila e segura. O aplicativo também auxilia os varejistas a encontrarem fornecedores e produtos de qualidade com as especificações desejadas.

Por exemplo, a busca por bananas orgânicas nanicas produzidas no Vale do Paraíba poderia ser uma tarefa muito difícil — tanto para o produtor encontrar alguém para vender, quanto para o comprador encontrar alguém para comprar. Através do nosso aplicativo, essa conexão se torna fácil e instantânea, gerando melhor lucratividade e disponibilidade para todos.
